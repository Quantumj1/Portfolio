# Osafo Builds - Personal Portfolio Website

![Osafo Builds](myportfolio/public/OB.png)

A modern, responsive personal portfolio website built with **React 19**, **Vite**, **Tailwind CSS 4**, and **React Router DOM**. Features smooth navigation, gradient animations, and a client-side contact form.

## ✨ Features
- 📱 Fully responsive design (mobile-first)
- 🧭 Client-side routing (Home, About, Projects, Resume, Contact)
- 🎨 Stunning gradient text effects and background images
- 📧 Contact form opens default email client pre-filled to `osafoparrypatrickjoseph@gmail.com`
- 📄 Resume download (PDF)
- ⚡ Fast HMR with Vite development server
- 🔍 ESLint + modern React best practices

## 🚀 Quick Start

### Prerequisites
- Node.js 18+

### Installation
```bash
cd myportfolio
npm install
```

### Development
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173)

### Build for Production
```bash
npm run build
npm run preview
```

### Linting
```bash
npm run lint
```
